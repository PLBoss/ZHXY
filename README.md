# 智慧校园管理系统

## 项目描述

这实现以年级，班级为单位，进行老师和学生信息记录和统计功能的信息管理系统

## 主要功能

注册与登陆，数据的增删查改，批量删除，数据的分页，模糊查询，用户上传头像，修改密码
